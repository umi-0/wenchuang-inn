# 🎬 视频嵌入指南（管理员用）

## 在哪里改

打开 `website/pages/cases.html`，找到 script 标签里的 `videos` 数组：

```javascript
var videos = [
  {
    id: 'v1',
    title: 'PS 零基础入门教程',
    desc: 'B站教程 · 认识工具栏...',
    src: 'bilibili',
    vid: 'BV1yT421k7SF',
    thumb: '🖥️'
  }
  // ← 在这里添加新视频
];
```

---

## B站视频

### 1. 获取视频 ID

打开 B站视频页面，看地址栏：

```
https://www.bilibili.com/video/BV1yT421k7SF/
                                ↑↑↑↑↑↑↑↑↑↑↑↑
                                这部分就是 bvid
```

### 2. 在 cases.html 中添加

```javascript
{
  id: 'v2',                              // 唯一编号，顺延即可
  title: '你的视频标题',
  desc: '简短描述，如：适合帆布包设计参考',
  src: 'bilibili',                        // 固定写 bilibili
  vid: 'BV1yT421k7SF',                   // ← 替换为实际 bvid
  thumb: '🎬'                             // emoji图标
}
```

---

## 抖音视频

### 1. 获取视频 ID

打开抖音视频页面或分享链接：

```
https://www.douyin.com/video/7123456789012345678
                              ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑
                              这串数字就是视频 ID
```

> 如果只有口令链接「7.12 xxxxx 复制打开抖音...」，先在浏览器打开口令，跳转后从地址栏取数字 ID。

### 2. 在 cases.html 中添加

```javascript
{
  id: 'v3',
  title: '你的视频标题',
  desc: '简短描述',
  src: 'douyin',                          // 固定写 douyin
  vid: '7123456789012345678',            // ← 替换为实际数字ID
  thumb: '🎬'
}
```

## 完整示例

假设要加两个视频（B站 + 抖音各一个）：

```javascript
var videos = [
  {
    id: 'v1',
    title: 'PS 零基础入门教程',
    desc: 'B站教程 · 认识工具栏、图层面板',
    src: 'bilibili',
    vid: 'BV1yT421k7SF',
    thumb: '🖥️'
  },
  {
    id: 'v2',
    title: '帆布包设计全过程',
    desc: '抖音 · 从草图到实物印刷',
    src: 'douyin',
    vid: '7123456789012345678',
    thumb: '🛍️'
  }
];
```

改完保存，刷新 `pages/cases.html` 页面即可看到新视频。

## 常见问题

| 问题 | 解决 |
|------|------|
| 视频显示"播放失败" | bvid 或抖音 ID 可能填错了，检查一下 |
| B站视频无法播放 | 部分 B站视频禁止外链嵌入，换个视频试试 |
| 抖音视频黑屏 | 抖音外链有时限，过期需要重新获取链接 |
| 想加更多视频平台 | 告诉我平台名称，我帮你加上对应的嵌入方式 |
