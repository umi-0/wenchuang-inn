## ✏️ 如何更新网站内容

### 1️⃣ 改文字 / 加链接（最常用）

1. 双击 `admin/index.html`
2. 输入密码登录
3. 顶部切换到要改的区域（软件 / 教程 / 素材 / 模板）
4. 点击卡片直接改标题、描述、链接
5. 改完点 **📤 导出 content.json**
6. 把导出的文件覆盖到 `data/content.json`
7. 刷新 `index.html` 看效果

### 2️⃣ 添加下载文件

1. 把文件放进 `files/` 文件夹
2. 在管理后台找到对应卡片，`下载链接` 填 `files/你的文件名`
3. 导出 → 覆盖 `data/content.json`

### 3️⃣ 同步到线上（Gitee Pages 开启后）

Git Bash 终端执行：

```bash
cd website/
git add .
git commit -m "更新内容"
git push
```
