## 📁 一、文件完整性——「三个文件，缺一个都不行」

- [ ] **效果图**（JPG 或 PNG 格式）
  > 效果图就是你的设计最终长什么样，给别人看的。相当于"成品展示照"。
- [ ] **PSD 源文件**（Photoshop 工程文件）
  > 这是你的"底稿"。像写 Word 要保留 .docx 一样，做图要保留 .psd。评审需要打开看你的图层结构和操作痕迹。
- [ ] **设计说明文档**（Word 或 PDF 格式）
  > 就是《文创设计投稿文件包》模板填完后导出的文件。
- [ ] 三个文件**全部齐全**——缺任何一个都会被退回，没有例外

## 📝 二、命名规范——「全部统一，一模一样」

- [ ] 三个文件的文件名**完全一致**，格式为：`专业班级+姓名+学号`
  > 例如：`电子信息2401+张三+2024001001.jpg`
  > 例如：`电子信息2401+张三+2024001001.psd`
  > 例如：`电子信息2401+张三+2024001001.docx`
- [ ] 分隔符用 `+`（半角加号），**不是**空格、不是 `_`、不是 `-`
- [ ] 压缩包命名也是同一个格式：`专业班级+姓名+学号.zip`
- [ ] 邮件主题是：`【文创投稿】专业班级+姓名+学号`
  > ⚠️ 注意括号是中文全角 **【】**，不是英文半角 [ ]

| 错误 | 为什么不行 |

## 🖼 三、效果图——「印出来不能糊」

- [ ] 格式是 **JPG** 或 **PNG**（不是 .psd、不是 .gif、不是别的东西）
- [ ] 分辨率 ≥ **300 dpi**
  > **怎么看？** PS 菜单：图像 → 图像大小 → 看"分辨率"一栏，数字必须 ≥ 300。
  > **为什么？** 低于 300dpi 印刷出来会模糊、有锯齿。这是印刷的最低门槛。
- [ ] 如果投稿的是印刷品（文件夹/卡套/帆布包/书签）→ 色彩模式设为 **CMYK**
  > **怎么设？** PS 菜单：图像 → 模式 → CMYK 颜色。
  > **为什么？** 屏幕显示用 RGB，印刷用 CMYK。RGB 直接印颜色会变灰变暗。
- [ ] 如果只是展示效果（不印出来），RGB 也可以
- [ ] 画面放大到 100% 查看——边缘没有锯齿、没有模糊、没有马赛克

## 🧩 四、PSD 源文件——「工厂能打开，评审能看到你的操作」

- [ ] **图层没有被全部合并**
  > **怎么检查？** 打开 PS → 右下角"图层"面板（没有就按 F7）→ 应该看到**多个图层**，而不是只有一层叫"背景"的东西。
  > **为什么？** 评审需要看到你的图层结构，确认你有自己操作过。全合并了就像交 Word 只交了一张截图——没办法证明是你做的。
- [ ] **文字图层没有栅格化**
  > **怎么检查？** 图层面板里，文字图层的缩略图右下角有一个小小的 **"T"** 字标记。有这个标记 = 安全。如果没有 = 文字已经变成普通图片了。
  > **为什么？** 工厂需要可编辑的文字来印刷。栅格化后文字会变模糊，工厂没法用。
  > **已经栅格化了怎么办？** 这次先提交，下次做图注意保留文字图层。在修改记录里备注一下。
- [ ] 如果用了 AI 生成的素材 → PSD 里**有你自己动手修改过的痕迹**
  > 比如：调了颜色、加了文字、重新排了版、替换了某个局部、手动修了某个区域。总之不能让评审看到"AI 出的图原封不动放在里面"。
- [ ] 所有外部图片素材已经**嵌入**文件
  > **什么意思？** 你从电脑里拖进 PS 的图片，如果不做嵌入，换个电脑打开就会显示红叉（找不到图片）。
  > **怎么嵌入？** PS 菜单：图层 → 智能对象 → 嵌入所有链接的对象。点一下就行。

## 🏭 五、生产可行性——「设计能真的做出来」

- [ ] **线条不会太细**
  > 特别是徽章、钥匙扣这种小物件。在 300dpi 下，线条粗细不要少于 3–4 像素。太细工厂刻不出来。
- [ ] **重要内容不贴边**
  > 印刷品（文件夹、卡套、帆布包）边缘会被裁切或缝合。院徽、文字、重要图案至少离边缘 **5mm** 以上。
- [ ] 徽章类：颜色控制在 3–5 种以内（颜色越多越贵、越容易出错）
- [ ] 卡套类：挂绳孔、摄像头孔等位置留空，没放重要图案
- [ ] 如果设计中有透明或镂空部分 → 已标注说明
